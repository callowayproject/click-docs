# admin

Admin commands.

**Usage:**

```text
admin [OPTIONS] COMMAND [ARGS]...
```

**Options:**

```text
  --help  Show this message and exit.
```

## reset

Reset the system to defaults.

**Usage:**

```text
admin reset [OPTIONS]
```

**Options:**

```text
  --force  Force the reset.
  --help   Show this message and exit.
```

## secret

Top-secret command.

**Usage:**

```text
admin secret [OPTIONS]
```

**Options:**

```text
  --help  Show this message and exit.
```

## status

Check system status.

**Usage:**

```text
admin status [OPTIONS]
```

**Options:**

```text
  --verbose  Enable verbose output.
  --help     Show this message and exit.
```
