# root

Root command with subgroups.

**Usage:**

```text
root [OPTIONS] COMMAND [ARGS]...
```

**Options:**

```text
  --help  Show this message and exit.
```

## admin

Admin commands.

**Usage:**

```text
root admin [OPTIONS] COMMAND [ARGS]...
```

**Options:**

```text
  --help  Show this message and exit.
```

## hello

Simple program that greets NAME.

**Usage:**

```text
root hello [OPTIONS]
```

**Options:**

```text
  --name TEXT  The person to greet.  [required]
  --help       Show this message and exit.
```
