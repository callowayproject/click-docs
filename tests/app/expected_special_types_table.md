# special-types

Command with various special types.

**Usage:**

```text
special-types [OPTIONS]
```

**Options:**

| Name | Type | Description |
| --- | --- | --- |
| --fmt | one of: json, yaml, toml | Output format. |
| --level | 0<=x<=10 | Log level. |
| --ratio | 0.0<=x<=1.0 | Ratio value. |
| --date | %Y-%m-%d, %d/%m/%Y | A date. |
| --infile | file (r) | Input file. |
| --msg | TEXT | Message. |
