# special-types

Command with various special types.

**Usage:**

```text
special-types [OPTIONS]
```

**Options:**

```text
  --fmt [json|yaml|toml]      Output format.
  --level INTEGER RANGE       Log level.  [0<=x<=10]
  --ratio FLOAT RANGE         Ratio value.  [0.0<=x<=1.0]
  --date [%Y-%m-%d|%d/%m/%Y]  A date.
  --infile FILENAME           Input file.
  --msg TEXT                  Message.
  --help                      Show this message and exit.
```
